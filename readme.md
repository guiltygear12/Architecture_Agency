## 반응형 웹사이트 포트폴리오
<hr>

##### 제작자: 박민형
##### 사용된 기술: HTML/CSS/JQUERY
##### 포트폴리오 컨셉트
> HTML+CSS+JQUERY 퍼블리싱으로 반응형 웹사이트

##### 주제 : 게임 < ELDEN RING > 을 소개하는 웹사이트 제작
##### 구성
> 페이지1 : 메인페이지
페이지2 : 게임의 특징
페이지3 : 주요 등장인물 소개
페이지4 : 지역별 특징 안내
페이지5 : 게임의 주요 스토리
페이지6 : 게임 주요 시스템 안내
페이지7 : 게임 구매안내
